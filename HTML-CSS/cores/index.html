Cores

Temos 4 tipos de cores no css

De forma escrita exemplo color:red

De forma hexadecimal exemplo colo: #ff0000

De forma hexadecimal exemplo colo: #f00

De forma RGB exemplo color: rgb(255,0,0)

De forma RGBA exmplo color: rgba(225,0,0,0.5)

De forma HSL exemplo color: hsl(0,100%,50%)

H = Hue(matriz e tonalidade)

S = Saturation (saturação)

L = Ligthness (Luminosidade ou brilho)

De forma HSLA exemplo color: hsla(0,100%,50%,50%)
